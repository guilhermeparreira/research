Objetivo¶

O objetivo deste tutorial discutir aspectos bsicos de anlise exploratria de dados atravs da linguagem python. Um conjunto de dados reais obtidos da iOS app store usado para exemplificao dos conceitos. Especificamente, vamos discutir os seguintes pontos

  1. Leitura e estrutura da base de dados.
    • Carregando arquivos .csv da web.
  2. Introduo anlise exploratria de dados
    • Tabelas de frequncia;
    • Medidas de tendncia central e disperso;
    • Grfico de pizza (pie chart);
    • Grafico de barras (bar plot);
    • Heat map.
    • Histrogramas;
    • Diagramas de disperso;
    • Boxplots.

Aplicativos da iOS app store¶

Este conjunto de dados refere-se a mais de 7000 Apple iOS mobile aplicativos disponveis na apple app store. Os dados foram extrados via web scrap usando iTunes Search API da Apple Inc website. Os dados esto disponveis na forma de dois arquivos .csv atravs da plataforma Kaggle no seguinte endereo Kaggle conjunto de dados I. Neste momento vamos trabalhar apenas com o arquivo AppleStore.csv que contm as seguintes colunas:

  • id: App ID;
  • track_name: Nome do aplicativo;
  • size_bytes: Tamanho em bytes;
  • currency: Tipo de moeda de venda;
  • price: Preo de venda;
  • rating_count_tot: Nmero de avaliaes dos usurios (todas as verses);
  • rating_count_ver: Nmero de avaliaes dos usurios (verso atual);
  • user_rating: Mdia das avaliaes dos usurios (todas as verses);
  • user_rating_ver: Mdia das avaliaes dos usurios (verso atual);
  • ver: Cdigo da ltima verso.
  • cont_rating: Categoria de contedo.
  • prime_genre: Genero primrio.
  • sup_devices.num: Nmero de dispositivos suportados;
  • ipadSc_urls.num: Nmero de screenshots para mostrar;
  • lang.num: Nmero de lnguas suportadas;
  • vpp_lic: Vpp Device Based Licensing Enabled

Como uma complementao a esta base de dados o arquivo appleStore_description.csv traz os seguintes dados:

  • id: App ID (unifica as bases);
  • track_name: Nome do aplicativo;
  • size_bytes: Tamanho em bytes;
  • app_desc: Descrio do aplicativo.

Para a importao correta da base de dados trs argumentos da funo read_csv so essenciais: Primeiro o sep que indica qual o caracter delimitador, neste caso ",". O argumento header indica qual linha da base de dados contm o nome das colunas. Em geral a primeira linha da base contm o nome das colunas. Entretando, lembre-se que o python comea seus ndices em zero, sendo assim, header = 0. Finalmente, o argumento index_col indica qual coluna contm o nome das linhas. Neste caso, novamente a primeira coluna e consequentemente index_col = 0. Estamos prontos para carregar a base de dados e fazer uma descrio rpida para verificar se a importao foi bem sucedida.

At o momento vimos as principais formas de apresentar variveis qualitativas nominais e ordinais, bem como, a relao entre elas. De forma similar, para apresentar a distribuio de variveis quantitativas discretas ou contnuas podemos usar grfico de barras (quantitativa discreta com poucos valores) e histogramas (quantitativas contnuas ou discreta com muitos valores). Para exemplificar, considere a varivel preo (price), cujo histograma apresentado abaixo.

In [16]:
dados1.hist(column='price')
Out[16]:
array([[<matplotlib.axes._subplots.AxesSubplot object at 0x7f77bcf116a0>]],

Infelizmente, o grfico no parece muito informativo. Isso devido ao fato da distribuio da varivel preo ser muito assimtrica, ou seja, temos muitos valores pequenos e apenas alguns valores elevados. Podemos, transformar a varivel preo para induzir simtria e ter uma melhor viso de sua distribuio. Uma possvel transformao o logaritmo.

In [17]:
dados1.hist(column='price', log = True)
Out[17]:
